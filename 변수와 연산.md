# 변수와 연산

## 변수

컴퓨터 메모리 어딘가에 저장되어 있는 객체를 참조하기 위해 사용되는 이름

객체 : 숫자, 문자, 클래스 등 값을 가진 모든 것

### 할당 연산자 =

변수를 할당하기 위한 연산자

id, type메서드

### 변수할당

```python
a = 2
b = c = 3
```

.

**pythonic**

```python
a,b = 1,2
print(a,b) # 1,2
a,b = b,a
print(a,b) # 2,1
```

### 변수연산

```python
a,b,c = 1,2,'hi'
print(a,b,c)  # 1 2 hi
print(a+b)    # 3
a += b
print(a)      # 3
print(a * b)  # 6
print((-b*b)/3) # -1.333333333
print(c + ' hello') # hi hello
print(c*3)    # hihihi
```
