```javascript
function compose(args) {
  return function (n) {
    let calValue = n;

    for (let i = args.length - 1; i >= 0; i--) {
      calValue = args[i](calValue);
    }

    return calValue;
  };
}
```

함수의 합성에 대한 문제

1. 이는 함수형 프로그래밍과 관련이 있다.
2. reduceRight 메서드
3. 컨텍스트를 보존하는 것에 대한 방법이 나온다. call, apply를 사용하는 것
