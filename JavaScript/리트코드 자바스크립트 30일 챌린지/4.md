```javascript
function map(arr, fn) {
  const initArr = [...arr];

  for (let i = 0; i < arr.length; i++) {
    initArr[i] = fn(arr[i], i);
  }

  return initArr;
}
```

콜백 함수에 대한 문제

1. 콜백을 사용하는 이유는, 재사용 가능한 코드를 구현하고 싶기 때문이다.
2. 메모리에 배열을 사전 할당하면, for ... of로 순회하거나 직접 push하는 것보다 빠르다.
3. 이보다 빠른 것은 인자를 건들이는 것이다. 하지만 이는 불변성의 문제가 있다.
